# Escopo ProviderX Planning Hub

## Conceito

O ProviderX Planning Hub é uma plataforma interna para planejar, estruturar, simular e apresentar o negócio ProviderX antes da escala comercial.

Ele responde perguntas como:

- Qual é o plano de negócios atual?
- Quais verticais compõem a ProviderX?
- Quais produtos pertencem a cada vertical?
- Qual dor de mercado cada solução resolve?
- Qual é o modelo de receita planejado?
- Quanto a ProviderX pode faturar em cenários diferentes?
- Quais custos e investimentos estão previstos?
- Qual estrutura operacional será necessária?
- Quais decisões estratégicas foram tomadas?
